# Student Result Comparison System

A Python project that compares the marks of two students and displays their performance using a graph.

## Features

- Accepts details of two students
- Takes marks for 5 subjects
- Calculates total marks
- Calculates percentage
- Compares student performance
- Displays graphical output using Matplotlib

## Technologies Used

- Python
- Matplotlib

## Installation

```bash
pip install matplotlib
```

## Run the Project

```bash
python student_result.py
```

## How It Works

1. Enter Student 1 details and marks.
2. Enter Student 2 details and marks.
3. The program calculates totals and percentages.
4. A graph is displayed comparing the marks of both students.
